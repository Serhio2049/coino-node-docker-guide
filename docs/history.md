# Coino Project History

## Timeline
- **2013-2016**: Original Scrypt-based coin
- **October 2016**: Relaunch with PoW Scrypt
- **June 20, 2017**: Migration to X11 (PoW) + PoS hybrid (version 2.0.0)
- **2026**: Docker image created by Serhii for preservation

## Version 2.0.0 Specifications
- **Algorithm**: PoW/X11 + PoS hybrid
- **Block time**: 365 seconds
- **PoW reward**: Starts at 8 CNO, halves every 10,000 blocks to 0.125 CNO
- **PoS reward**: 0.5% annual (minimum stake age 30 days, maximum 90 days)

## Network Information
- **RPC Port**: 29299
- **P2P Port**: 29293
- **Active nodes**: ~11 (as of 2026)

## Preservation Note
This documentation and Docker image were created to preserve the ability to run a Coino v2.0.0 node, as the original source code is no longer publicly available.
